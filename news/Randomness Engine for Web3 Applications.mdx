## **⚙️ How RANDAO Works**
RANDAO operates through four main phases:

1. **📝 Commit Phase**  
   Participants submit hashes of their secret numbers

2. **🔓 Reveal Phase**  
   Participants gradually disclose their secret numbers

3. **🧩 Aggregation Phase**  
   The final random number is generated from Hash of all values

4. **🎯 Final Output**  
   - Creates a single random value that:
     - 🚫 Cannot be predicted beforehand
     - 🤖 Cannot be manipulated afterward
     - 👁️ Can be publicly verified

**✨ Key Advantages:**
- 🌐 Requires no external oracles
- ⛓️ Fully on-chain process
- 💰 More cost-efficient than alternative solutions
